Passo a passo para instalação do RemoveSing

1 - Executar o arquivo "python-3.13.2-amd64.exe";

2 - Entrar na pasta da aplicação e criar uma atalho para o arquivo "(aplicação).py";

3 - Ir nas propriedades do atalho criado e adicionar a variável "pythonw" na opção "Destino".

4 - Testar o uso da aplicação.

Passo a passo atualização/instalação GIT

1 - Instalar o Git;

2 - Crie uma pasta onde os arquivos serão armazenados, por exemplo, "C:\PyProjects".

3 - Abra o Git Bash ou terminal e execute: git clone https://github.com/renannunesz/PyProjects.git C:\PyProjects

4 - Execute o arquivo de atualização para os diretorios
